# FitQuest

An interactive fitness adventure platform that gamifies physical activity for children, making exercise fun and engaging through AI-powered challenges, parent-child activities, and progress tracking.

## Features

- 🎮 **Adventure-based Learning**: Children progress through fitness chapters with engaging storylines
- 🤖 **AI-Powered Challenges**: Dynamic fitness challenges generated based on activity level
- 👨‍👩‍👧‍👦 **Parent-Child Games**: Interactive activities that families can do together
- 📸 **Photo Gallery**: Capture and share fitness journey moments
- 📊 **Progress Tracking**: Visual progress indicators and achievement system
- 🔒 **Parental Controls**: Screen time management and activity monitoring
- 🎖️ **Rewards System**: Earn gems and unlock shop items

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun
- Supabase account (for backend services)

### Installation

```sh
# Clone the repository
git clone https://github.com/Aditya17-04/FitQuest.git

# Navigate to the project directory
cd FitQuest

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies Used

- **Frontend**: React + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn-ui
- **Backend**: Supabase
- **Authentication**: Supabase Auth
